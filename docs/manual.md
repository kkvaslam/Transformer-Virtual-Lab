# Transformer Lab (Part 2): Efficiency vs Load Resistance
This virtual lab replicates the PHYS‑194 transformer experiment.

## Aim
Study transformer efficiency as a function of resistive load.

## Procedure
1. Select a resistance value.
2. Record Vp, Ip, Pin, Vs, Is, Pout, and efficiency.
3. Repeat for all R values.
4. Plot efficiency vs R.
5. Determine the value where efficiency is maximum.
